# .github

This repository contains organization-wide GitHub configuration for the Breaking Boot organization.

## Contents
profile/README.md — public organization profile shown on the Breaking Boot homepage.

## Updating the Organization Profile

The public Breaking Boot homepage is generated from:

[profile/README.md](profile/README.md)

Changes to that file will appear on Breaking Boots's main GitHub page.

## Maintainers

Breaking Boot administrators are responsible for reviewing changes to this repository.
